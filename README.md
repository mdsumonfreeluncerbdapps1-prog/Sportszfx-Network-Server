BDApps Cricket Live Score SMS Server

A Node.js based SMS service for BDApps that provides live cricket match updates through SMS menus.

Features

- Live Matches
- Upcoming Matches
- Recent Matches
- Match Score Details
- SMS Pagination

How It Works

User sends SMS keyword to the BDApps shortcode.
The server processes the request and returns cricket match information via SMS menu.

Technology

- Node.js
- Express.js
- Axios API requests
- BDApps SMS Listener

Deployment

1. Upload the project to GitHub
2. Deploy to a Node.js hosting service (Render / Railway / VPS)
3. Set the BDApps SMS endpoint:
